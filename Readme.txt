Το πρόγραμμα είναι υλοποιημένο στη γλώσσα C, σε περιβάλλον linux.

Για να τρέξει σωστά, απαιτείται να υπάρχουν οι φάκελοι που πρόκειται να αντιγραφούν από τον server στον client.

Στο αρχείο sript_client.sh - που έχει φτιαχτεί για τη διευκόλυνση στην εκτέλεση του προγράμματος, ο βασικός φάκελος είναι ο folder1, ο οποίος περιέχει κάποια αρχεία και τον folder2 (ο οποίος έχει κι αυτός κάποια αρχεία).
